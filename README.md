用户文档：流程图编辑器功能指南
一、引言
欢迎使用我们的流程图编辑器，这是一款功能强大且易于使用的工具，旨在帮助用户高效地创建、编辑和管理流程图。本文档将详细介绍软件的主要功能、操作方法及使用技巧，帮助您快速上手并充分发挥其潜力。
二、功能列表
1、文件操作
打开文件：支持打开fc工程文件，并可直接进入编辑状态。
另存文件：提供将流程图另存为PNG、JPG、SVG等格式的功能，并支持自动打开最近			一次保存的地址，方便用户快速访问。
2、编辑功能
复制、粘贴、剪切：用户可轻松复制、粘贴、剪切单个图元，通过右键菜单或快捷键（如			Ctrl+C、Ctrl+V、Ctrl+X）进行操作。
图元堆叠调整：允许用户设置图元的堆叠关系，即前置或后置，以满足复杂的流程图布			局需求。
撤销重做：支持撤销（Ctrl+Z）和恢复（Ctrl+Y）操作，让用户能够灵活调整编辑过程中			的每一步。
3、绘图工具
绘图元素：提供矩形、圆形、线、箭头等多种基本绘图元素，用户可通过左侧图元库选			择并在画布上绘制。
图形的平移、缩放：支持鼠标拖动进行移动，选中图元后可通过节点调节其大小。
批量设置：可按住Shift键进行框选，对选中的图元进行批量设置颜色、位置等操作。
注释和标签
注释功能：每个图元均可设置文字内容，同时提供单独的文本框对象，方便用户添加注释或说明。
4、元素库
国标图元：支持国标中的20种图元，用户可根据需要选择合适的图元进行绘制。
图元属性：图元支持缩放、拖动、设置颜色、文字等属性，满足不同的绘制需求。
5、属性编辑
允许用户编辑元素的详细信息，如字体大小、样式、颜色等，使流程图更加美观和专业。
6、画布设置
视图比例：可设置画布视图比例，包括50%、75%、100%等多种选项，便于用户查看和		调整流程图。
网格线：提供黑白棋盘、蓝白棋盘、无网格线等多种网格线样式，帮助用户精确对齐图元。
7、交互功能
图元连线：支持图元之间的连线操作，通过拖拽节点连接点即可快速完成连线。
自吸附功能：在进行连线操作时，鼠标靠近图元连接点附近时会自动吸附，提高操作效率。
8、高级功能
文本查找/替换：支持全图的文字查找和替换功能，方便用户快速定位和修改文本内容。
导入背景图片：允许用户导入常见图像格式作为画布背景，增加流程图的可视化效果。
三、快捷键
为提升操作效率，编辑器提供了一系列快捷键。以下是部分常用快捷键的说明：
Ctrl+C：复制
Ctrl+V：粘贴
Ctrl+X：剪切
Ctrl+Z：撤销
Ctrl+Y：恢复
Ctrl+R/L：顺时针/逆时针旋转图元
Delete：删除图元
